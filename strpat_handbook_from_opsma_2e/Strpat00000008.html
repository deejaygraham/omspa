Overview

This handbook presents strategies and patterns, ready-to-apply practical advice for building effective object models. These strategies and patterns span these major categories:

Strategies for building object models

- Activities and model components - Establishing responsibilities

- Identifying purpose and features - Working out dynamics with scenarios

- Selecting objects - Discovering new strategies and patterns

Patterns for building object models

- Fundamental pattern - Plan patterns

- Transaction patterns - Interaction patterns

- Aggregate patterns

As you discover new strategies and patterns, please send them to Peter Coad (coad@oi.com) for future versions of this handbook. Your input will be gratefully acknowledged. Thank you for your help in advancing this important work.
