Interaction patterns

Interaction patterns are patterns of how objects work with each other. Overlay these patterns on players in other patterns.

The interaction patterns are:

Pattern #25. "Peer-Peer" Pattern

Pattern #26. "Proxy - Specific Item" Pattern

Pattern #27. "Publisher-Subscriber" Pattern

Pattern #28. "Sender - Pass Through - Receiver" Pattern

Pattern #29. "Sender-Lookup-Receiver" Pattern

Pattern #30. "Caller - Dispatcher - Caller Back" Pattern

Pattern #31. "Gatekeeper-Request-Resource" Pattern

Patterns for building object models
